# AIQuizlet — iOS

## О проекте

AIQuizlet — мобильное приложение для автоматической генерации тестов по учебным материалам. Студент фотографирует страницу конспекта или вставляет текст лекции, приложение отправляет материал в нейросеть и возвращает готовый тест с вопросами и вариантами ответов. Пользователь проходит тест, получает результат с разбором ошибок и сохраняет его в историю для повторения.

## Функциональность

- Регистрация и вход по email
- Создание теста через фото с камеры или из галереи
- Создание теста через ввод текста вручную
- Генерация вопросов с вариантами ответов через нейросеть
- Прохождение теста с подсветкой правильных и неправильных ответов
- Пояснения к каждому вопросу
- История всех пройденных тестов с датой и результатом
- Статистика в профиле
- Поддержка тёмной и светлой темы
- Поддержка русского и английского языков

## Технологии

- Язык: Swift 5+
- Минимальная версия iOS: 17
- UI: UIKit (верстка через код)
- Архитектура: MVVM + Coordinator
- Сеть: Moya
- Локальное хранение: SwiftData
- Бэкенд: Firebase
- Нейросеть: OpenRouter API (бесплатные vision модели)
- Верстка: SnapKit
- Codestyle: SwiftLint
- Локализация: SwiftGen
- Менеджер зависимостей: SPM
